# How to use it

The article [Hands-On Data Augmentation: Essential Techniques for Computer Vision with TensorFlow](https://medium.com/@thomas.zilliox/hands-on-data-augmentation-essential-techniques-for-computer-vision-with-tensorflow-794337c6bea1)
is a good example on how to use that toolkit.

But first you will need to clone the repo

```
!git clone https://github.com/ThomasPRZilliox/ml_utils.git
```

Then you can use it as follow:

```
import ml_utils.tf.computer_vision as cv
cv.random_crop_resize(image,cropped_factor=0.9)
```
